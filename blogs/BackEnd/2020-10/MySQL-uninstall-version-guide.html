# MySQL 免安装版配置

为什么使用免安装版？

  • 绿色无污染
  • 便携轻量
  • 灵活可控

# 开始

  1. 前往 MySQL 官网下载免安装版 MySQL Community Downloads

  2. 下载解压至任意目录，这里解压到 C:/MySQL 为例，解压后得到以下目录结构

  3. 配置环境变量：

新增系统变量
变量名：%MYSQL_HOME%
变量值：C:/MySQL

添加PATH：%MYSQL_HOME%\bin

检查是否设置成功命令行键入 mysql --version 如果显示失败，检查环境变量的设置

  1. 在 C:/MySQL/mysql-8.0.21-winx64/ 下新建 my.ini 文件（可选，如无特殊需求不建议自行配置）
[mysqld]
# 设置mysql的安装目录
basedir=C:\MySQL\mysql-8.0.21-winx64
# 设置mysql数据库的数据的存放目录，必须是data
datadirC:\MySQL\mysql-8.0.21-winx64
# mysql端口
port=3306
# 字符集
character_set_server=utf8
# 更多设置参数请自行查询文档
  1. 使用管理员权限打开 CMD 执行安装命令（如果忽略 .ini 文件，一切都按默认进行配置）
mysqld -install
  1. 执行 MySQL 初始化命令，在 mysql-8.0.21-winx64 目录下就会生成 data 文件夹
mysqld --initialize-insecure

  1. 启动 MySQL 服务
net start mysql
  1. 命令行访问数据库
mysql -u root -p
  1. mysql 正常语法
show databases;
  1. 设置 root 密码
mysqladmin -u root -p abc password 123456

使用 quit 或 exit 退出 mysql 命令符，再执行修改密码操作 如无设置密码可忽略 -p 参数

# 其余控制命令

关闭 MySQL 服务

net stop mysql

卸载 MySQL

mysqld -remove

查看服务状态 (mysql命令)

status;
